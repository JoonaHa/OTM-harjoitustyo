## Komentorivitoiminot

Ohjelman JUnit-testit voi suorittaa komennolla:
```
mvn test
```
Testeistä voi luoda kattavuusraportin komennolla:

```
mvn jacoco:report
```
Raportti on html-dokumentti _target/site/jacoco/index.html_ jota voi tarkastella selaimella.
